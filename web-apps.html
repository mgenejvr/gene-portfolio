= Web App Projects =

Upcycling Waste

It is a web-based application that has two user types: admin and client. The scope of the project focuses on developing a web-based application that would help the users of the system to disseminate essential information about upcycling recycled materials and food leftovers and gain knowledge about the ideas that are being disseminated. Its features will showcase the categorization for foods and crafts, recycling guidelines, comment section, chat, sharing to social media, and craft ideas and food ideas with instructions, needed ingredients or materials, images, and videos.
It is developed using PHP, HTML, CSS, Bootstrap, JavaScript, MySQL, and PHPMailer.
